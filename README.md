# ZTCars

### Installation

```sh
$ npm install
```

### Run it

1. Install the dependencies 
2. Start the server with
    ```shell
    $ node server
    ```
3. Then check address: `localhost:1337`

### Dependencies

Compatible with versions:

| Name | version |
| ------ | ------ |
| Mongodb | 3.4.1 |
| Express | 4.15.2 |
| Angular | 1.6.4 |
| Node js| 6.10.2 |
| Mongoose | 4.9.5 |
